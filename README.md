# 기후기술 분야 국제협력 전략지도 (v13)

정적 데모 버전입니다.

## 포함 기능
- 우선구축국 10개국 선택
- 38대 기후기술 선택
- 전략정보 대시보드(시각화 카드 + 상세 팝업)
- GIS 기반 우선지역 분석(기후리스크 히트존, 잠재수요권, 협력 연계축)
- 우선지역별 포인트 상세 검토
- 단계별 협력전략 상세 검토
- AI 통합검색 예시
- 요약보고서 HTML 다운로드

## 실행
```bash
npm install
npm run dev
```
